Extrapolación de Richardson para Mejorar la Derivada Numérica

La extrapolación de Richardson mejora la precisión de la derivada numérica al combinar dos aproximaciones con diferentes tamaños de paso.

Primero, calcula la derivada numérica D(h) usando un paso h: 

                    
                        D(h) = (f(x + h) - f(x)) / h

Luego, calcula la derivada D(h/2) con un paso más pequeño h/2: 

                    
                        D(h/2) = (f(x + h/2) - f(x)) / (h/2)

Usa la extrapolación de Richardson para combinar estas dos estimaciones y reducir el error principal: 

                    
                        Dextrapolado = (4/3) * D(h/2) - (1/3) * D(h)

Esta fórmula elimina el término de error principal, proporcionando una estimación más precisa de la derivada en el punto x.

La formula de D(h1) varia dependiendo el metodo de diferenciacion finita escogido

API del Método de Extrapolacion de Richardson


Para utilizar el método de Extrapolacion de Richardson, se debe realizar una petición POST al siguiente endpoint:

https://analisis-numerico-sand.vercel.app/metodos/diferenciacion/richardson


El cuerpo de la petición debe contener un objeto JSON con la siguiente estructura:



                    {
                        "funcion":"funcion",
                        "metodo":"nombre del metodo",
                         "version":numero de version del metodo,
                          "orden":orden de la derivada a calcular,
                          "h":valor de h,
                          "xi":valor de x a evaluar,
                          "nivel":nivel de extrapolacion,
                    }

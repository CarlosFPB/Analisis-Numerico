Explicación del Método de Newton Recursivo

El método de Newton Recursivo es un método de interpolación que se obtiene a partir de la fórmula de Newton. Este método se utiliza para encontrar un polinomio que pase por un conjunto de puntos dados. El polinomio resultante es de la forma:

P_n(x) = f[x_0] + f[x_0,x_1](x-x_0) + f[x_0,x_1,x_2](x-x_0)(x-x_1) + ... + f[x_0,x_1,...,x_n](x-x_0)(x-x_1)...(x-x_{n-1})


Donde los coeficientes \(f[x_0,x_1,...,x_k]\) se calculan de forma recursiva utilizando la fórmula:

f[x_0,x_1,...,x_k] = \frac{f[x_1,...,x_k] - f[x_0,...,x_{k-1}]}{x_k - x_0}


Para encontrar el polinomio interpolante, se evalúa la función en los puntos

\(x_0, x_1, ..., x_n\)

y se calculan los coeficientes \(f[x_0], f[x_0,x_1], ..., f[x_0,x_1,...,x_n]\) de forma recursiva.


El polinomio resultante es el polinomio interpolante de Newton, que pasa por los puntos dados.

API del Método de Newton Recursivo

Para utilizar el método de Newton Recursivo a través de la API, debes realizar una petición POST al siguiente endpoint:

https://analisis-numerico-sand.vercel.app/metodos/interpolacion/newton_recursivo


Los parametros se deben enviar en formato JSON, Existen dos formas para poder realizar el calculo:


1. Ingresando la función y los puntos unicamente de X en donde se evaluará la función para obtener los puntos de Y

Ejemplo:

                        {
                            "tipo": 1,
                            "funcion": "3*x**3 + 5*x**2 - 4*x + 1",
                            "matrizPuntos": [
                                ["0", "1", "2", "3"]
                            ]
                        }

2. Ingresando los puntos de X y Y sin la función

Ejemplo:

                        {
                            "tipo": 2,
                            "matrizPuntos": [
                                ["0", "1", "2", "3"],
                                ["-1", "6", "31", "18"]
                            ]
                        }

API del Método de Trazadores Cubicos

Para utilizar el método , debes realizar una petición POST al siguiente endpoint:

https://analisis-numerico-sand.vercel.app/metodos/integracion/Metodo-multipleTabla


El cuerpo de la petición debe contener un objeto JSON con la siguiente estructura:


                    {
                        "matrizPuntos":[[x0, y0,..xn], [x1, y1..yn]],
                    }

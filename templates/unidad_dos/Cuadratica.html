API del Método de la Cuadrática


Para utilizar el método de la Cuadrática, se debe realizar una petición POST al siguiente endpoint:

https://metodos.herokuapp.com/metodos/directos/gradomas2/Cuadratica


El cuerpo de la petición debe contener un objeto JSON con la siguiente estructura:



                    {
                        "funcion": "x**2 + 2*x - 3"
                    }
